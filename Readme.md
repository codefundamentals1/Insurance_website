


# Installation

1. Clone the repository:

```
git clone https://github.com/codefundamentals1/Insurance_website.git

Navigate to the project directory:

cd Insurance_website 

Install dependencies:

npm install

Usage
Development

Start the client (React) development server:

       npm run dev

 Start the server (Express) production server:

     npm start


Technologies Used

    React
    Tailwind css
    Express.js
    Nodejs
    MySQL
    Vite
